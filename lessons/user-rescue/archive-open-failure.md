# 压缩包打不开？先别急，试这 3 步

## 你可能遇到的是：

- 双击没反应
- 提示"文件损坏"
- 解压到一半失败
- 文件名乱码
- AI / 小龙虾读不了压缩包

---

## 先试这 3 步

### 1. 先把文件移到桌面

不要放在很深的文件夹里。
路径太长、中文太多，有时会导致解压失败。

### 2. 换 7-Zip 打开

不要只用系统自带解压。
推荐用 [7-Zip](https://www.7-zip.org/) 重新打开一次。

### 3. 重新下载一次

如果提示"文件损坏"，很可能是下载不完整。
重新下载后再试。

---

## 还不行？

不用解释太多，直接发这 3 样：

1. 报错截图
2. 文件后缀：.zip / .rar / .7z / 不知道
3. 你用的是 Windows / Mac / 不知道

不会描述也没关系，发截图就行。

## 给维护者/Agent 的结构化记录

- **Root cause:** 路径过长、中文路径编码、系统解压工具兼容性、下载不完整、文件本身损坏
- **Fix:** 移动到短路径 + 换 7-Zip + 重新下载
- **Prevention:** 小龙虾应自动识别格式、检查损坏、提示装 7-Zip、中文路径/超长路径给明确提示
